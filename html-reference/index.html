HTML Reference Page

HTML Elemenet

The HTML tag is vital for your code. You put it at the start and end of every code to say that your code is html. It looks like < html >. You use it in every single html code. It doesn't display.

Title Element

A title is used to title the tab of the webpage. It is represented by < title >. It is placed in the head. It labels the tab of the webpage. This one is HTML Reference.

Heading Element

The head element is used to put in everything not visual such as the title of the webpage. It is shown as < head >. You use it if you want to label your website after the html tag and before the body tag.

Body Element

The body element is what is visually displayed on the website. It is showed using the < body > tag. It is placed after the title tag and is in the html tag.

Heading Element (This is a heading tag visually)

A Header is used to title your paragraphs and come in 6 different sizes. They are put in the body. Headers are basically bigger font sized paragraphs. It is represented by < h1 >. It is mosly used before the paragraph tag to signify what the paragraph tag is going to be about.

Paragraph Element

A Paragraph element is used to write paragraphs. They are put in the body. All of these sentences are part of a paragraph and is shown as < p >. It usually comes after the heading tag.

Break Element

Breaks are used to indent a line.
As you can see, this line is under the previous sentence. They are put in the body. It is shown as < br >. It is used to put things in the next line and can be used whenever.

Line Element

Lines are used to break apart different ideas. They are put in the body. All the horizontal lines are made using the line element. Shown as < hr >. It is used to break apart different ideas.

Comment Element

Comments are messages you write in code that don't affect the code and don't show. They can be put anywhere. It looks like < !-- -->. You can't actually see these in the webpage.

Link Element

Links are used to put in a link into phrases, images, or words. They are put in the body. They are represented by using < a >. They can direct you to another place in your website or any website on the web.

This is a link to Google

Image Element

Images are used to just put in images. It's represented by < img >. They are put in the body and you can resize them.

List Element

Lists are used to list a group of items. They are put in the body and can be either bullet points or numbers. Shown by < ul >, < ol >, or < li >.

Tables are used to create tables of stuff and compare them. Tables put things in a grid and are mostly used in the body. It looks like < table >.

Block Element

Blocks are used to put a certain syle such as font size or type to a block of stuff. It is used by the code < div >. It is used in the body to change the style of a certain part of lines.
